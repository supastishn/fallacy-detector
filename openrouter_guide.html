Setting Up Your OpenRouter API Key

OpenRouter.ai provides a unified interface to access various Large Language Models (LLMs). To use OpenRouter with this application, you'll need an API key and the correct base URL.

Step 1: Create an OpenRouter Account

  • Go to https://openrouter.ai.
  • Sign up for a new account or log in if you already have one.

Step 2: Fund Your Account

  • OpenRouter operates on a pre-paid credit system. You'll need to add funds to your account to use the API for paid models. Some models may be available for free or with a free trial quota.
  • Navigate to the "Billing" or "Credits" section in your OpenRouter dashboard and add credits if necessary.

Step 3: Get Your API Key

  • Once logged in, navigate to the "Keys" section (usually found under your account settings or API settings).
  • Click on "Create Key" or a similar button.
  • Give your key a descriptive name (e.g., "FallacyDetectorApp").
  • Copy the generated API key. Treat this key like a password and keep it secure.

Step 4: Configure in This Application

  • Go to the API Key Settings page in this application.
  • Paste the copied OpenRouter API key into the "API Key" field.
  • In the "Base URL" field, enter: https://openrouter.ai/api/v1
  • You can leave the "Default Model" field blank if you want OpenRouter to use its default, or enter a specific model available on OpenRouter that you wish to use by default (e.g., openai/gpt-4o, anthropic/claude-3-opus, mistralai/mistral-7b-instruct). Check the OpenRouter documentation for available model identifiers.
  • Click "Save Settings".
  • You can use the "Test API Connection" button on the settings page to verify your setup.

Important Notes:

  • Model Availability: Not all models are free. Check the pricing on OpenRouter for each model you intend to use.
  • Free Model Recommendations (check OpenRouter for current availability and terms):
    • Examples of potentially free models (text-only, verify on OpenRouter):
      • deepseek/deepseek-r1-0528:free
      • deepseek/deepseek-r1:free
      • deepseek/deepseek-chat-v3-0324:free
    • For models with image support, check OpenRouter's model list for current free options.
    • meta-llama/llama-4-maverick:free is a free model for image support
    • Model names, capabilities, and availability can change; always verify on OpenRouter.ai.
  • Rate Limits: Be aware of any rate limits OpenRouter might impose on your account or specific models, especially free tiers (e.g., free models often have a daily limit).
  • Security: Your API key is stored in your browser's local storage. Clear application storage via browser settings if you are on a shared computer and have finished your session.

That's it! You should now be able to use the application with models via OpenRouter.ai.
